# wallet-manager
simple wallet manager using django
# How to use?
## step 1:
```
$ git clone https://github.com/Ml06py/wallet-manager.git
```
## step 2:
```
$ cd wallet-manager
```
## step 3:
```
$ python manage.py makemigrations
```
## step 4:
```
$ python manage.py migrate
```
## step 5:
```
$ python manage.py runserver
```
